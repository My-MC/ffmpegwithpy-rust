# ffmpegwithpy

This registry is a Rust rewrite of [ffmpegwithpy](https://github.com/My-MC/ffmpegwithpy).

## to Develop

### Install Dependence

``` bash
pip install -r requirements.txt
```

### Develop Build

``` bash
task dev
```

### Build Release Wheel

``` bash
maturin build --release -o dist
```
